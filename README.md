# Ftour Lyoum - Android App

Made by louay

## 🚀 How to get the APK (the easiest way)

1. Create a new repository on GitHub.
2. Upload ALL files from this ZIP into that repository.
3. Go to the "Actions" tab at the top of your GitHub repo.
4. On the left, click "Build APK".
5. Click the "Run workflow" button (on the right).
6. Wait 2-3 minutes.
7. When it finishes, click the run, scroll to the bottom, and download "app-debug-apk".
8. Put that APK on your Android phone and tap it to install.

## 💻 Alternative: build it yourself in Android Studio

1. Open this folder in Android Studio.
2. Let Gradle finish syncing.
3. Click Build > Build APK(s).
4. The APK will be in app/build/outputs/apk/debug/.
